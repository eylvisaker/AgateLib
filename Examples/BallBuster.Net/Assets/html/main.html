Ball: Buster
Break bricks, grab power ups, and keep the ball from falling off the screen. Ball: Buster is a fast paced "Breakout" style game that's simple enough to pick up and play, but thorough enough to include a built in level editor.

Game Features:
• Hardware acceleration for great performance
• 20 Unique Power Ups
• An unlimited amount of balls on screen
• Built in Level editor
• Level files are human readable and easily shared
• Particle and alpha effects. Rotating and zooming effects.
• Fast, Fun Gameplay
• Open Source
• "Attract" mode where the computer demonstrates how fun the game is!
