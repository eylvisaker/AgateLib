Installation Directions
Installation is easy, and chances are, if you're reading this; you've already done so. If you've chosen to download the installer/uninstaller package, just double click and the installer will guide you through the process. Upon completion shortcuts to play Ball: Buster and read this manual will be found in your start menu and desktop. In the case of downloading the manual install, just unzip all of the files to the desired location, and double click app.exe to play. For Operating Systems other than windows, please check for suitable instructions within the Ball: Buster root directory.
Uninstalling Ball: Buster
Why? It's not like it's hurting you! Well, okay, if you really want to delete it, just select the uninstaller from the start menu, and it will be erased from your hard drive. If you did a manual install, simply delete the folder you installed Ball: Buster to.
