Options menu

• Start game begins the game
• Level editor begins the level editor
• Quit will exit the program
• Play Music will change music on or off
• Scrolling background will determine if the background scrolls
• Fullscreen/windowed will change video modes
• VSync on/off is for "power users". It should be kept on by default.
• Up and Down arrow keys allow you to change what level you start playing on. The computer will demonstrate gameplay on that level in the background.
 

If you wait for five seconds without moving the mouse, the title screen will vanish so you can watch the computer play. Move the mouse to get back to the title screen.
